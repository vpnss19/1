SS，全称为 Shadowsocks（下文简称为SS），是一种基于 Socks5 代理方式的网络数据加密传输包的工具，可以使用其于科学上网 也就是翻墙 ，是目前最稳定，最常用的科学上网工具之一，全平台支持

SSR，全称为 ShadowsocksR（下文简称为SSR） 来历：

ShadowsocksR是breakwa11发起的Shadowsocks分支，在Shadowsocks的基础上增加了一些数据混淆方式，称修复了部分安全问题并可以提高QoS优先级。后来贡献者Librehat也为Shadowsocks补上了一些此类特性，甚至增加了类似Tor的可插拔传输层功能。

ShadowsocksR开始时曾有过违反GPL、发放二进制时不发放源码的争议，不过后来ShadowsocksR项目由breakwa11转为了与Shadowsocks相同的GPL、Apache许可证、MIT许可证等多重自由软件许可协议。

2017年7月19日，ShadowsocksR作者breakwa11在Telegram频道ShadowsocksR news里转发了深圳市启用SS协议检测并被大量用户转发，引发恐慌。7月24日，breakwa11发布了闭源的SS被动检测程序，引发争议。 2017年7月27日，breakwa11遭到自称 “ESU.TV” 的不明身份人士人身攻击，对方宣称如果不停止开发并阻止用户讨论此事件将发布更多包含个人隐私的资料，随后breakwa11表示遭到对方人肉搜索并公开个人资料的是无关人士，为了防止对方继续伤害无关人士，breakwa11将删除GitHub上的所有代码、解散相关交流群组，停止ShadowsocksR项目。但项目已被多人fork。并有人在其基础上继续发布新的版本，例如SSRR

火箭 Shadowrocket
“火箭”通常指的是 Shadowrocket 的一种通俗叫法。Shadowrocket 是一款用于科学上网的代理工具，能够帮助用户访问被限制的网站和服务。它支持多种协议，提供灵活的配置选项，是许多用户在移动设备上实现网络安全的首选应用。

Shadowrocket 香港
使用位于香港的代理节点可以帮助用户访问特定的内容，尤其是在访问某些地区限制的网站时。用户可以在 Shadowrocket 中添加香港的代理节点，以获得更快的连接速度和更好的访问能力。

Shadowrocket 自建
用户可以选择自建代理服务器，通常使用 VPS（虚拟专用服务器）来搭建 Shadowsocks 或 V2Ray 服务器。自建服务器的好处在于用户可以完全控制服务器性能和安全性，避免使用公共节点带来的隐私风险。搭建完成后，用户只需在 Shadowrocket 中配置相应的服务器信息即可。

Shadowrocket 限制
在某些地区，使用 Shadowrocket 可能会面临访问限制。这可能是因为当地网络审查或代理服务被封锁。用户可以尝试更换节点、使用不同的协议，或者调整应用设置来绕过这些限制。

Shadowrocket UDP
Shadowrocket 支持 UDP 协议，这对于某些应用（如游戏或视频流）非常重要，因为这些应用通常依赖于 UDP 进行更快的数据传输。在设置节点时，用户可以选择支持 UDP 的代理，以提高某些应用的性能和响应速度。

Shadowrocket Safari
用户可以在 Safari 浏览器中使用 Shadowrocket 提供的代理服务，但需要确保在 Shadowrocket 应用中正确配置了代理。通过 Shadowrocket 的设置，用户可以实现 Safari 浏览器的流量转发，访问被限制的网站。
